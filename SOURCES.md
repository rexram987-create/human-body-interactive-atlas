# Sources, image credits, and medical disclaimer

## Medical content

The atlas is an introductory educational resource. It is not medical advice and must not be used for diagnosis or treatment.

The anatomical summaries should be reviewed against:

- OpenStax, *Anatomy and Physiology 2e* (2022):  
  https://openstax.org/details/books/anatomy-and-physiology-2e/
- MedlinePlus Medical Encyclopedia, U.S. National Library of Medicine:  
  https://medlineplus.gov/encyclopedia.html
- NCBI Bookshelf, U.S. National Library of Medicine:  
  https://www.ncbi.nlm.nih.gov/books/

Content requiring clinical interpretation should be reviewed by a qualified healthcare professional before publication.

## Images

The current interface uses historical anatomical illustrations described by the project as public-domain material. Before production release, every image must have an individual record containing:

- Original title and creator, when known
- Publication date
- Source page URL
- License or public-domain rationale
- Date retrieved
- Any modifications made by this project

The project README references *Internal Organs of the Human Body from The Household Physician* (1905), available through Wikimedia Commons. The repository currently does not contain enough per-file provenance to verify that every image came from that work, so the images must not be described collectively as verified public domain until those records are completed.
