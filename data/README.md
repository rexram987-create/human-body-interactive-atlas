# Data Folder

## עברית

התיקייה הזו מיועדת לקובצי מידע עתידיים של האטלס.

בשלב הנוכחי המידע על המערכות והאיברים נמצא בקובץ:

`js/app.js`

בהמשך ניתן להעביר את המידע לקובצי JSON נפרדים, למשל:

- `digestive.json`
- `respiratory.json`
- `circulatory.json`
- `nervous.json`
- `skeletal.json`
- `muscular.json`
- `urinary.json`
- `sense-organs.json`
- `male-reproductive.json`
- `female-reproductive.json`

---
