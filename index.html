אטלס גוף האדם

Human Body Interactive Atlas

כל טאב מציג איור אנטומי נפרד המתאים למערכת הגוף שנבחרה, עם אזורים לחיצים והסברים דו־לשוניים.

Each tab displays a separate anatomical illustration for the selected body system, with clickable areas and bilingual explanations.

אפשר ללחוץ על הנקודות שבתמונה או על שמות האיברים שמתחתיה.

You can click the markers on the image or the organ names below it.

תמונות נפרדות

Separate Images

כל טאב טוען איור ייעודי למערכת הגוף שלו.

Each tab loads a dedicated illustration for its body system.

אזורים לחיצים

Clickable Areas

לכל תמונה יש נקודות לחיצה ורשימת איברים משלה.

Each image has its own clickable markers and organ list.

מבנה מסודר

Organized Structure

הקוד הופרד לקבצי CSS ו־JavaScript כדי שיהיה קל להרחיב את האתר.

The code is separated into CSS and JavaScript files so the site can grow easily.
